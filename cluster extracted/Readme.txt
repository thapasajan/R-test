Notes:
1. Currently, the clustering is being done text(.txt) files only.
2. This software is for Windows OS only

Instruction for running document Clustering script
1. Double-click respective batch(.bat) file for running either clustering with Gibbs or with VEM.
2. Give address of your directory which contains .txt files that you want to cluster.
  If you want to cluster on provided sample documents, give the full address of "sampleDocs" folder. 
3. Specify the number of cluster you want.
4. The result are stored as .csv files in the location you provided in step 2.
